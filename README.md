## What is MedPath AI?

MedPath AI is a full-stack conversational healthcare navigation platform built for the Indian market. Describe your symptoms in plain language вҖ” the AI figures out what procedure you need, finds the right hospitals in your city, shows a personalised cost breakdown adjusted for your medical history, and walks you through applying for a Poonawalla Fincorp medical loan. All in one chat.

No phone calls. No clinic hopping. No guesswork.

---

## Features

- **Conversational clinical intake** вҖ” Gemini 2.5 Flash extracts procedure, city, urgency, and ICD-10 codes from plain language
- **Emergency bypass** вҖ” detected at вүҘ85% confidence, skips clarification entirely and surfaces hospitals immediately with a red alert banner
- **Intelligent hospital ranking** вҖ” scored by cost match, quality rating, NABH/JCI accreditation, wait time, and GPS distance
- **Personalised cost breakdowns** вҖ” comorbidity multipliers (diabetes, cardiac, kidney, etc.) applied on top of procedure base costs
- **PFL loan pre-qualification** вҖ” instant GREEN / YELLOW / RED eligibility decision based on CIBIL, income, FOIR, and loan multiple
- **Gemini document extraction** вҖ” upload your salary slip or CIBIL report and the AI fills in your financial profile automatically
- **Real-time loan tracking** вҖ” patient app polls loan status every 3 seconds; sees the officer's decision the moment it's made
- **5-step registration wizard** вҖ” health profile, comorbidities, insurance, financials, and emergency contact
- **Spectator-safe officer dashboard** вҖ” completely separate Vite build for PFL officers; ships zero patient code

### Coming Soon
- **AI second opinion** вҖ” challenge a hospital recommendation with a follow-up question
- **User profiles** вҖ” persistent health history and past consultation summaries
- **Real PFL API** вҖ” live Poonawalla Fincorp loan disbursement integration

## Getting Started

### Prerequisites
- Node.js 20+
- Python 3.11+
- A Supabase project (free tier works)
- A Google Gemini API key (Gemini 2.5 Flash access required)

### 1. Clone the repo
```bash
git clone https://github.com/yourusername/MedPathAI.git
cd MedPathAI
```

### 2. Backend вҖ” Python environment
```bash
python -m venv venv
source venv/bin/activate          # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Backend вҖ” environment variables
Create a `.env` file inside `backend/`:
```env
SUPABASE_URL=https://<your-project>.supabase.co
SUPABASE_KEY=<your-supabase-anon-or-service-key>
GEMINI_API_KEY=<your-gemini-api-key>
DOCUMENT_BUCKET=medical-documents
DOCUMENT_EXTRACTION_ENABLED=false
```

### 4. Run the backend
```bash
python ./backend/main.py   # вҶ’ http://localhost:8000
```

### 5. Frontend
```bash
cd medpathai
npm install
npm run dev         # вҶ’ http://localhost:5173 (patient app)
npm run dev:pfl     # вҶ’ http://localhost:5174 (PFL officer dashboard)
```

### 6. Verify everything is running
- `http://localhost:5173` вҶ’ Patient chat app
- `http://localhost:5174` вҶ’ PFL officer dashboard
- `http://localhost:8000` вҶ’ `{"status": "MedPath AI is running рҹҡҖ"}`
- `http://localhost:8000/docs` вҶ’ FastAPI Swagger UI (test `/api/chat` here)

On first run the backend loads hospital, procedure, and city data from Supabase into memory. Seed CSVs are in `backend/data/`.

## How It Works

1. Register with your health profile, comorbidities, insurance details, and financials
2. Upload financial documents вҖ” salary slip, ITR, CIBIL report (Gemini extracts the data automatically)
3. Open the chat and describe your symptoms or the procedure you need
4. The AI asks up to **3 targeted clarifying questions** вҖ” then commits to a recommendation
5. Get a ranked list of hospitals in your city with personalised cost breakdowns
6. Select a hospital and apply for a Poonawalla Fincorp medical loan in one click
7. The AI runs an instant eligibility check and submits your application
8. Track your loan status live вҖ” the PFL officer's decision appears in seconds

## ML Scoring Pipeline

Every argument вҖ” sorry, every *argument* in the chat вҖ” runs through four stages in the LangGraph pipeline:

| Node | Model / Engine | Role |
|---|---|---|
| Intent | Gemini 2.5 Flash | Extracts procedure, city, budget, urgency, emergency flag, ICD-10 hint |
| Provider | Pure Python + Pandas | Scores hospitals by cost, quality, accreditation, wait time, GPS distance |
| Cost | Formula Engine | Personalised breakdown with comorbidity multipliers + PFL loan options |
| Response | Gemini 2.5 Flash | Generates warm, clinically-phrased summary or emergency alert |

**Comorbidity multipliers:** Kidney Disease +22% В· Cardiac +18% В· Diabetes +12% В· Obesity +10% В· Hypertension +8% В· Asthma +6%

**PFL eligibility tiers:** CIBIL вүҘ700, income вүҘвӮ№15k/month, FOIR вүӨ50%, loan вүӨ10Г— income вҶ’ **GREEN** (instant approval) В· soft flags вҶ’ **YELLOW** В· hard failure вҶ’ **RED** (alternatives suggested)
